# Sangeeta Dhami - Professional Portfolio Website 🍃✨

Welcome to the source code repository of my professional portfolio website! This project is a modern, responsive showcase of my skills, experience, projects, and achievements in Artificial Intelligence, Machine Learning, Web Development, and Research.

## 🎨 Theme & Styling
- **Professional Royal Deep Blue Theme:** Clean, corporate styling with deep navy and vibrant blue tones.
- **Glassmorphism:** Elegant semi-transparent container cards with a blurred background overlay.
- **Dark/Light Mode:** Includes a theme toggler that persists choices via `localStorage`.
- **Animated Gradients & Micro-interactions:** Smooth animations on scroll, floating backdrop particles, and hover effects on components.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla ES6)
- **Icons:** FontAwesome (via CDN)
- **Typography:** Google Fonts (Inter & Roboto)
- **Form Handling:** FormSubmit AJAX integration for direct email delivery

## 📂 Project Structure
- `index.html` - Core structure and content
- `style.css` - Custom styling rules, variables, keyframes, and themes
- `script.js` - Scroll animations, theme toggle, and contact form handling
- `assets/` - Directory for images and documents (e.g., Profile Photo and Resume)
